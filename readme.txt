Add any pages, posts or products in any page related to any category or tag using shortcodes

== Description ==
Add any pages, posts or products in any page related to any category or tag using shortcodes. A built in shortcode generator is also added in this plugin, so you dont have to remember any shortcode.

<h3>Features</h3>
<ol>
	<li>Shortcode Generator</li>
	<li>Single Post</li>
	<li>Multiple Posts related to Category</li>
	<li>Multiple Posts related to Tag</li>
	<li>Single Page</li>
	<li>Single Product</li>
	<li>Multiple Products</li>
	<li>Responsive Grid System from 1 to 6 Columns</li>
	<li>Posts count per shortcode</li>
	<li>Show/Hide Author</li>
	<li>Show/Hide Tags</li>
	<li>Show/Hide Date</li>
	<li>Show/Hide Thumbnail</li>
	<li>Show/Hide Pagination</li>
	<li>AngularJS Based</li>
</ol>

== Installation ==
1. Go to plugins in your dashboard and select \'add new\'
2. Search for \'WP Query\', install it and activate
3. Now click on WP Query menu page and generate shortcode
4. Copy and paste shortcode in page, post or even in widget.
